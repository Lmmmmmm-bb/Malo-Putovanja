## 技术栈

- Vite
- Vue3 + Vue-Router4 + Pinia
- Axios
- ESLint + Prettier
- Commit Lint + Husky
- Element Plus + Tailwind CSS
- SCSS + SCSS-Typed-Modules
- pnpm

## 启动项目

```shell
# 开发调试
> pnpm install # 安装依赖
> pnpm dev # 启动开发服务器
# 生产环境
> pnpm build # 构建生产环境
> pnpm preview # 本地预览
```
